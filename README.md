# 🎬 Mavrex Movie Manager

Mavrex Movie Manager is a responsive movie collection and tracking application built with Vanilla JavaScript, HTML, and CSS. It allows users to organize, manage, and review their favorite movies through an intuitive and modern interface.

## ✨ Features

* Add new movies with details such as:

  * Movie title
  * Poster image URL
  * Genre
  * Release year
  * Rating
  * Personal notes
  * Watch status

* Edit existing movie information

* Delete movies with confirmation prompts

* View detailed movie information in a dedicated modal

* Search movies instantly by title

* Separate movie categories:

  * All Movies
  * Watchlist
  * Watched

* Form validation for user input

* Local Storage persistence (data remains after page refresh)

* Responsive design for desktop, tablet, and mobile devices

* Custom notification system for user actions

* Dynamic page updates without requiring manual refreshes

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage API
* Font Awesome Icons

## 📂 Project Workflow

1. User adds a movie through the Add Movie form.
2. Movie data is validated before submission.
3. Valid movie data is stored in Local Storage.
4. Movie cards are generated dynamically and displayed instantly.
5. Users can:

   * Search movies
   * View details
   * Edit movie information
   * Delete movies
6. All changes automatically update the UI and Local Storage.
